# Bakery_ARM
This project applies **Association Rule Mining** to analyze customer purchasing habits. Insights help optimize:
- Product arrangement
- Product recommendations
- Inventory management
- Ad optimization

## Library
```install.packages(c("arules","arulesViz","dplyr","ggplot2"))```
